# waterpipe 資料完整性盤點 — 對照工作清單

> 自動產出。資料源：`index.html` 內嵌 DB（執行 `python scripts/extract-db.py` 重抽）。
> `null` 不必然是缺漏；表名宣告的多角度 / 多形式則是硬合約，缺一就是漏。

範圍：12 張表 / 720 筆 row

## 使用方法

對照原圖（`images/T{id}_*.webp`）逐項判定，把每個 `- [ ]` 改成：

- `- [x]` ← JDPA 本來就沒做（不必補），不必補充說明
- `- [!]` ← 確認漏掉，請在該項下面**附上規格圖上的數值**，例如 `D=900 22.5°: R=___, L1=___, L2=___, 管心長_l=___, 質量=___, T=___`

對完傳回給 Claude，會批次補進 `index.html` DB 並提 PR。
